##  ABC Analysis Tool (Python)
Скрипт для автоматизации категорийного анализа продаж. Классифицирует товары по вкладу в выручку (группы A, B, C), что помогает оптимизировать складские запасы.

### Функционал
*   **Автоматизация:** Заменяет ручную работу в Excel, исключая ошибки.
*   **Обработка данных:** Загрузка из CSV и расчет кумулятивной доли выручки.
*   **Интеграция:** Экспорт результатов в SQLite для дальнейшей визуализации.

### Стек технологий
*   Python 3.x
*   Pandas, NumPy
*   SQLite3
