ABC Analysis Tool (Python + Pandas + SQLite)
Описание: Скрипт для автоматизации категорийного анализа продаж. Классифицирует товары по вкладу в выручку (группы A, B, C), что позволяет оптимизировать складские запасы и закупочную политику.
Стек: Python 3.x, Pandas, NumPy, SQLite3.
Функционал:
Загрузка данных из CSV-файлов.
Расчет кумулятивной доли выручки и присвоение категорий.
Экспорт результатов в реляционную БД (SQLite) для дальнейшей визуализации или интеграции с другими системами.
Зачем это нужно: Заменяет ручную работу в Excel, исключает ошибки в расчетах.
